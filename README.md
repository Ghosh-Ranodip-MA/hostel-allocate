# UEM-HOSTEL | Advanced Roommate Matching & Allocation

A powerful, streamlined Hostel Management System designed to handle student data, automate room allocation using professional matching criteria, and provide a visual dashboard for wardens.

## 🚀 Features

- **Smart Roommate Matching:** Automatically pairs students using key criteria:
  - Gender isolation (Separate Male/Female hostels)
  - Academic Stream matching
  - Food habit compatibility (Veg/Non-Veg)
  - Year of study parity
  - Native Language matching
- **Warden Command Center:**
  - Full hostel configuration (Floors, Rooms, Gender rules)
  - Bulk student data import via CSV
  - Visual room layout with real-time occupancy tracking
  - Manual room assignment & overrides
- **Student Resident Portal:**
  - Secure login via name and phone
  - View allocation details and roommate records
- **Data Portability:** Export the final allocation registry to CSV for offline usage.

## 🛠️ Technology Stack

- **Structure & Logic:** HTML5 & Vanilla JavaScript
- **Styling:** Vanilla CSS (Modern aesthetic with Rajdhani & DM Sans typography)
- **Data Persistence:** LocalStorage-based data management for instant use.

## 📦 Getting Started

This is a **Zero-Dependency** application. You don't need any complex setup:

1. Download or clone this repository.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari).
3. Start managing your hostel!
